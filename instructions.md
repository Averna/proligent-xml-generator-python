# Instructions

Here are instructions for AI working on this repository.

- Line endings are CRLF.
- The coding language is python.
- The 'docs/xsd' folder is read-only. These files are generated from an external source and copied here manually.
- The python source code should be in the 'src' folder.
- Datetime written to XML shall follow this format: `ISO 8601 combined date and time representation with a numeric timezone offset.`
- The field `DataSourceFingerprint` is optional but we will always generate it. It is a randomly generated GUID.
